# 대리자

## 사용 방법

1. Visual Studio에서 `Chapter40-Delegates.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice-01 | 교안 코드 따라치기 - 대리자 기초, 매개변수, 멀티캐스트, 내장 대리자, 익명 메서드 (11개 과제) |
| CodingPractice-02 | 교안 코드 따라치기 - 인스턴스 메서드, 멀티캐스트, 배열 변환 (3개 과제) |
| CodingPractice-03 | 교안 코드 따라치기 - 내장 대리자 활용, 익명 메서드 멀티캐스트 (2개 과제) |

## 연습문제

각 프로젝트의 README에 있는 문제를 읽고 코드를 작성합니다.

| 프로젝트 | 문제 |
|----------|------|
| MathOperation | 사칙연산 대리자 |
| MulticastDelegate | 멀티캐스트 대리자 |
