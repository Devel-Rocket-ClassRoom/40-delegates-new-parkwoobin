# 사칙연산 대리자

정수 두 개를 받아 정수를 반환하는 대리자를 사용하여 사칙연산 프로그램을 작성하시오.

**요구사항:**

1. `MathOp`라는 이름의 대리자를 선언하시오
   - 매개변수: `int` 2개
   - 반환 타입: `int`

2. 다음 사칙연산 메서드를 구현하시오
   - `Add(int a, int b)` → a + b
   - `Subtract(int a, int b)` → a - b
   - `Multiply(int a, int b)` → a * b
   - `Divide(int a, int b)` → a / b

3. 각 메서드를 대리자에 할당하여 테스트하시오
   - 테스트 값: a=20, b=4

## 예상 실행 결과

```
=== 사칙연산 대리자 ===

[덧셈]
20 + 4 = 24

[뺄셈]
20 - 4 = 16

[곱셈]
20 * 4 = 80

[나눗셈]
20 / 4 = 5
```
